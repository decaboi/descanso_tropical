Sitio Web para Hoteles y Posadas. 

Ideal para esos establecimientos de servicio de Hospedaje en la Isla de Margarita, que no tienen presencia en Internet a traves sde un sitio web odicial,  y solo tienen vaga presencia en redes sociales como Instagram. 

Desarrollado en Herramientas como:  HTML, CSS, JAVASCRIPT.
Posee procesos de automatización como un  CHATBOT  con IA que interactua con el usuarios y le responde las dudas que tenga para que pueda reservar sin problema y se sienta seguro.
